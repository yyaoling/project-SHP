### 2.params和query参数的区别

1. query 传统问号传参
url 格式：xxx.com/product?id=123
模板内获取数据：this.$route.query.id 
2. params 动态路由匹配
url 格式：xxx.com/product/123
模板内获取数据：this.$route.params.id
注意这个方式参数字段名 id 要在路由配置中定义 用冒号的形式标记
参数可以继续拼接 /student/:id/:name/:age/:address
他必须严格按照 url 的配置格式访问
